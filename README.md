# 거래소별 시총 순위 보드

[index.html](C:\Users\wlvht\Documents\Codex\2026-04-22-spot-usdt\index.html) 를 브라우저에서 열면 됩니다.

`file://` 로 열면 거래소 API가 차단될 수 있습니다. 그 경우 `server.ps1` 로 로컬 서버를 띄운 뒤 `http://127.0.0.1:8791` 로 여는 방식이 더 안정적입니다.

포함된 기능:

- 바이낸스 `USDT` 현물 시총 순위
- 업비트 `KRW` 마켓 시총 순위
- 빗썸 `KRW` 마켓 시총 순위
- 100개 단위 페이지 이동
- 동시상장 종목 필터
  - 바이낸스 + 업비트 + 빗썸
  - 바이낸스 + 업비트
  - 바이낸스 + 빗썸

데이터 방식:

- 거래 가능 종목 목록은 각 거래소 공개 API 기준입니다.
- 시총은 CoinGecko 유통량과 각 거래소 현재가를 곱해서 계산합니다.
- 동시상장 여부는 심볼 기준으로 판별합니다.

주의:

- 같은 심볼을 여러 프로젝트가 공유하는 경우 CoinGecko 매칭 오차가 생길 수 있습니다.
- `file://` 환경에서 브라우저 보안 정책이나 거래소 CORS 정책 때문에 일부 API 호출이 막힐 수 있습니다.
